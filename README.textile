h1. Qt::JRuby DSL

h2. Description

Give a little sugar to Qt::JRuby.

h2. Requirements

* Java 1.5+
* Qt Jambi 4.4+
* JRuby 1.1.3+
* Qt::JRuby Core 0.2.1+

h2. Installation

<pre><code>git clone git://github.com/nmerouze/qtjruby-dsl.git
cd /path/to/qtjruby-dsl
jruby -S rake</code></pre>

or

<pre><code>jruby -S gem install nmerouze-qtjruby-dsl --source http://gems.github.com</code></pre>

h2. Example

<pre><code>require 'qtjruby-dsl'

Qt.app do
	window :id => 'main' do
		button('Click Me!').clicked { alert 'Clicked!' }
	end
end</code></pre>
